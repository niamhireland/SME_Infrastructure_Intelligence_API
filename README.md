## About The Project

This project is a backend API that simulates how a Solutions Architect would assess a small or medium-sized business and recommend suitable technology infrastructure.
It takes basic company information such as size, location, and number of remote workers, enriches it using free public APIs and applies a rule-based decision engine to generate structured infrastructure recommendations.

The goal is to demonstrate API integration, backend system design and practical solution architecture thinking.

### Technologies Used

- Python
- FastAPI
- OpenStreetMap Nominatim API
- REST Countries API

## Project Purpose 

This project was built to develop practical experience with:
- API development and integration
- backend system design
- data enrichment pipelines
- rule-based decision systems
- solutions architecture thinking (business → technical mapping)

It is in active, early development.
